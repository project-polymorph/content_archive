## SOME 的帮助 { data-search-exclude }

SOME 的淋浴设施一年中的每一天都可供使用。将提供洗浴用品和毛巾。SOME 为男性和女性提供标准的早间时段。跨性别者、有医疗转诊者、残疾人和其他经授权的紧急情况可以在下午预约淋浴。

前台还提供卫生用品包，包括肥皂、洗发水、乳液、除臭剂、牙刷和牙膏、剃须刀和女性卫生用品。
